# Данные о такси в Нью-Йорке 
Данные о такси в Нью-Йорке включают более 3 миллиардов поездок на такси и арендованных транспортных средствах (Uber, Lyft и т. Д.), Совершенных в Нью-Йорке с 2009 года.
